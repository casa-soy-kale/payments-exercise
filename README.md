# Casa Coding Exercise

## Setup

1. Clone this repo
1. Backend
	1. Navigate to the `backend` directory
	1. Run `yarn install`
	1. Add features to `backend/router.js`
1. Frontend
	1. Navigate to the `frontend` directory
	1. Run `yarn install`
	1. Add features to `frontend/App.jsx`
